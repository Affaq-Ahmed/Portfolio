## Overview

Access Point is a unified platform that provides **centralized access to multiple
third-party services** through API integrations. Instead of wiring up each
provider separately, it brings Google, Meta, HubSpot, TikTok, and Microsoft
together behind one authentication and orchestration layer.

## My Role & Contributions

I led this project as **Tech Lead**, owning its direction and delivery rather
than its hands-on implementation. Under my lead, the team built:

- **Provider integrations** — Google, Meta, HubSpot, TikTok, and Microsoft.
- **Authentication workflows** — handling auth across the different providers.
- **API orchestration** — a unified layer coordinating calls across services.

## Key Features

- **Unified integrations** across Google, Meta, HubSpot, TikTok, and Microsoft.
- **Cross-provider authentication** workflows.
- **API orchestration** behind a single platform.

## Tech Stack & Architecture

- **App:** Next.js with Supabase.
- **Integrations:** Google, Meta, HubSpot, TikTok, and Microsoft APIs, unified
  behind one orchestration and auth layer.

## Challenges & Solutions

- **Many providers, one layer** — each third-party service has its own auth and
  API quirks; the work was unifying them behind a single, consistent
  orchestration layer.
- **Authentication across providers** — building auth workflows that handle the
  differences between each provider cleanly.
